## 🚀 Getting Started

### Prerequisites
- [MySQL](https://dev.mysql.com/downloads/) (v8.0 or above recommended)
- Any SQL client: MySQL Workbench, DBeaver, or terminal

### How to Run

```bash
# Connect to MySQL
mysql -u root -p

# Run a specific file
source /path/to/Day1.sql;
```

Or import directly in **MySQL Workbench**:
`File → Open SQL Script → Select file → Execute`

## 📚 Topics Covered

- **DDL** — Data Definition Language (`CREATE`, `ALTER`, `DROP`, `TRUNCATE`, `RENAME`)
- **DML** — Data Manipulation Language (`INSERT`, `UPDATE`, `DELETE`)
- **DQL** — Data Query Language (`SELECT`, filters, sorting)
- **DCL** — Data Control Language (`GRANT`, `REVOKE`)
- **TCL** — Transaction Control Language (`COMMIT`, `ROLLBACK`, `SAVEPOINT`)
- **Joins** — INNER, LEFT, RIGHT, FULL OUTER
- **Subqueries** — Nested & correlated queries
- **Aggregate Functions** — `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
- **Group By & Having**
- **Views, Triggers, Stored Procedures**
- **Indexes**
